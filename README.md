# Portfolio v2055 - AI Systems Engineer

A futuristic 2055-concept portfolio website built with React, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack

- **Framework:** Vite + React 19 + TypeScript
- **Styling:** Tailwind CSS 4.x with custom 2055 theme
- **Animations:** Framer Motion
- **Fonts:** JetBrains Mono (technical), Inter (body)
- **AI Widget:** Transformers.js ready

## 🎨 Design Philosophy - 2055 Aesthetic

### Visual Elements

| Element | Description |
|---------|-------------|
| **Quantum Grid** | Animated perspective grid rotating in 3D space |
| **Neural Network** | Pulsing nodes representing AI connections |
| **Holographic Panels** | Glass-like panels with scan-line effects |
| **3D Tilt Cards** | Mouse-reactive perspective transforms |
| **Neon Glow** | Bioluminescent accents (cyan, magenta, lime) |

## 🛠 Development

```bash

cd portfolio-v2

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
